# marksman-lsp

Markdown language server plugin for Claude Code, powered by [marksman](https://github.com/artempyanykh/marksman).

## Supported Extensions

`.md`, `.markdown`

## Binary Required

`marksman` — already installed in the container image at `/usr/local/bin/marksman`.

## Install Plugin

```
/plugin install /claude-here/plugins/marksman-lsp
```
